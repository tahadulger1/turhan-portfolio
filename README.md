# Emirhan Turhan Portfolyo

Bu proje Emirhan Turhan'ın yazılım ve tasarım çalışmalarını sergilediği profesyonel portfolyo sitesidir.

## Özellikler

- **Modern Tasarım**: Motion (framer-motion) ile pürüzsüz animasyonlar.
- **Dinamik İçerik**: SQLite veritabanı ile yönetilen projeler.
- **Admin Paneli**: Proje ekleme, silme ve düzenleme imkanı.
- **Responsive**: Tüm cihazlarla tam uyumlu (Mobil, Tablet, Masaüstü).
- **SEO Dostu**: Optimize edilmiş meta etiketleri ve semantik HTML.
- **Erişilebilirlik**: WCAG standartlarına uygun navigasyon ve tasarım.

## Kurulum

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. `.env` dosyasını oluşturun ve `ADMIN_PASSWORD` değerini belirleyin.

3. Uygulamayı çalıştırın:
   ```bash
   npm run dev
   ```

## Teknolojiler

- React 19
- Vite
- Tailwind CSS
- Express
- SQLite (Better-sqlite3)
- Lucide React
- Motion
